# digigarden
A soft, aesthetic productivity web app that turns your daily habits into a growing garden. Stay consistent, earn points, and watch your plant evolve as you complete tasks and maintain streaks.

---

## Features

### Garden Growth System

* Visual plant growth based on daily progress
* 45-day journey with evolving stages (seed to tree to bloom)
* Daily login rewards and streak tracking

### Streaks and Points

* Earn points for:

  * Daily login (+5)
  * Completing tasks (+5 each)
* Maintain streaks by logging in consistently
* Bonus rewards at milestones (7, 15, 30, 45 days)

### Task Manager

* Add daily tasks
* Mark tasks as complete
* Tasks reward points and contribute to growth

### Notes (Secret Notebook)

* Create and store personal notes
* Grid-style note display
* Click to view full content

### Pomodoro Timer

* Custom work and break durations
* Animated hourglass interface
* Sand color customization
* Alerts when session ends

### Profile Management

* Edit username
* Add profile picture using URL
* Change password

### User Interface

* Soft pastel theme
* Dark mode toggle
* Smooth animations and transitions
* Mobile-friendly layout

---

## Tech Stack

* HTML5 – Structure
* CSS3 – Styling and animations
* JavaScript (Vanilla) – Logic and interactivity
* LocalStorage – Data persistence without backend

---

## How to Run

1. Download or clone the project
2. Open the HTML file in your browser
3. Start using the app

No installation or server setup required.

---

## Data Storage

* All user data is stored locally in the browser using localStorage
* No external database or authentication system is used
* Data will be lost if browser storage is cleared
